# A. 자연어 템플릿 모음

이 부록은 팀 지식베이스에 바로 복사해 쓸 수 있는 자연어 템플릿이다. 프로젝트에 맞게 이름과 항목을 줄여도 된다.

## 작업 헌장

```text
# 작업 헌장

작업 이름:

배경:

목표:

비목표:

대상 범위:

건드리면 안 되는 것:

완료 조건:

검증 방법:

보고 형식:
```

## 인터뷰 요청

```text
너는 이 프로젝트의 명세 인터뷰어다.
아직 해결책을 제안하지 말고, 요청의 숨은 가정과 모호한 용어를 드러내줘.

진행 방식:
1. Wonder 질문으로 문제 공간을 넓힌다.
2. Ontology 질문으로 핵심 용어를 좁힌다.
3. 답변이 필요한 질문을 우선순위 순서로 7개 이하로 정리한다.

요청:
...
```

## 컨텍스트 패킷

```text
# 컨텍스트 패킷

반드시 읽기:

필요하면 읽기:

읽지 말기:

주의할 프로젝트 규칙:

오래되었거나 불확실한 정보:
```

## 실행 세션 시작

```text
이 프로젝트는 ai-harness/를 팀 공용 하네스로 사용한다.
먼저 README, PROJECT_CONTEXT, AI_WORKFLOW, EVAL_RUBRIC을 읽어줘.
아직 작업하지 말고, 이번 세션에서 지켜야 할 규칙만 요약해줘.
```

## 검증 요청

```text
작업 결과를 EVAL_RUBRIC과 작업 헌장의 완료 조건으로 검토해줘.
증거가 없는 항목은 "미확인"으로 표시하고 통과 처리하지 마.
수정이 필요한 항목은 중요도 순서로 정리해줘.
```
